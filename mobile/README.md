# Expo mobile app (experimental)

This directory contains the default **Expo / React Native** scaffold. It is **not** the production technician app.

Technicians should use the **Next.js PWA** at `/mobile/*` in the `frontend/` project:

- Install from the browser (Add to Home Screen)
- Routes: `/mobile/dashboard`, work orders, inspections, schedule, roadside, etc.

Do not add shop features here until the PWA roadmap is complete and native delivery is explicitly prioritized.
